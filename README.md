## Café Miskatonic

Um site fictício de cafeteria com temática cósmica e lovecraftiana, desenvolvido com foco em prática de HTML, CSS e primeiros passos com JavaScript.

## Sobre o projeto

O **Café Miskatonic** é uma landing page estilizada que apresenta uma cafeteria “extradimensional”, com um visual inspirado no horror cósmico de Lovecraft.

O objetivo do projeto foi simular um cenário mais próximo do mundo real, com:

* Estrutura de páginas
* Seções bem definidas
* Prazo de entrega
* Identidade visual consistente

Mesmo sendo um projeto simples, ele marca um avanço importante na organização de código e construção de layout.

---

## Funcionalidades

* Navegação por seções (Home, Sobre, Cardápio, Entrega, App, Contato)
* Área de destaque (Hero Section)
* Cardápio com itens estilizados
* Botões interativos com JavaScript
* Estilização temática com foco em identidade visual

---

## Tecnologias utilizadas

* **HTML5** → Estrutura do site
* **CSS3** → Estilização e layout
* **JavaScript (básico)** → Interações simples (eventos de clique)

---

## Design

O projeto segue uma estética:

* Dark / horror cósmico
* Tons de roxo, rosa e amarelo
* Elementos visuais inspirados em espaço e criaturas lovecraftianas
* imanges editadas para o projeto

A proposta foi equilibrar:

> um visual estilizado e interessante sem tornar o desenvolvimento excessivamente complexo.
